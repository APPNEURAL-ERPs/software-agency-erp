# GatewayOS

Gateway routing, traffic control, request validation, authentication gateway, service gateway, tenant gateway, edge gateway, and policy enforcement.

## Owns

- Gateway
- GatewayRoute
- GatewayService
- GatewayPolicy
- TrafficRule
- RateLimit
- GatewayTrace
- GatewayIncident

## Run

```bash
npm install
npm run build
npm start
```

Default service URL: `http://localhost:10714`.

Catalog: `../README.md`.
