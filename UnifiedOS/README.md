# UnifiedOS

Cross-OS orchestration that connects all AppNeuroX OS modules into one registry, dependency graph, workflow layer, event layer, and command center.

## Owns

- OSModule
- OSDependency
- CrossOSWorkflow
- OSEventRoute
- UnifiedCommand
- HealthSignal
- ImpactReport
- IntegrationMap

## Run

```bash
npm install
npm run build
npm start
```

Default service URL: `http://localhost:10717`.

Catalog: `../README.md`.
