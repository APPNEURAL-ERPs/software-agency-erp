# VendorOS

Vendor, supplier, contractor, service provider, onboarding, evaluation, contracts, purchase, payment, risk, compliance, and vendor relationship management.

## Owns

- Vendor
- VendorContact
- VendorDocument

## Run

```bash
npm install
npm run build
npm start
```

Default service URL: `http://localhost:10718`.
