# EntityOS

Universal business object definitions, schemas, records, relationships, lifecycle, ownership, validation, search, audit, and metadata.

## Owns

- EntityType
- EntityRecord
- EntitySchema
- EntityField
- EntityRelationship
- EntityLifecycle
- EntityValidation
- EntityIndex

## Run

```bash
npm install
npm run build
npm start
```

Default service URL: `http://localhost:10712`.

Catalog: `../README.md`.
