# Software Agency ERP

Software Agency ERP for managing operations, workflows, records, billing, and reports.

## Overview

This repository is the APPNEURAL starter space for a software agency product. It is intended to hold the product roadmap, data model, implementation notes, and future application source code for the ERP.

## Core Modules

- Client accounts
- Project delivery
- Campaign tasks
- Retainers
