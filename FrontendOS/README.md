# FrontendOS

Frontend architecture, app shells, design systems, components, pages, layouts, routing, state, forms, accessibility, performance, and app experience.

## Owns

- FrontendApp
- AppShell
- Page
- Route
- Component
- Layout
- Theme
- Form
- AccessibilityCheck

## Run

```bash
npm install
npm run build
npm start
```

Default service URL: `http://localhost:10713`.

Catalog: `../README.md`.
